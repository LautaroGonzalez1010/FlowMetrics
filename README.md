📊 Flow Metrics
.
.
Flow Metrics es una aplicación web moderna diseñada para la gestión, visualización y análisis de métricas comerciales en tiempo real.

El objetivo del proyecto es ofrecer una experiencia clara, rápida e intuitiva para el seguimiento de ventas, clientes y rendimiento general, todo desde un dashboard elegante y optimizado.

🚀 ¿Qué es Flow Metrics?
.
.
Flow Metrics es una plataforma tipo SaaS orientada al análisis de datos comerciales.
Permite visualizar información clave del negocio en un solo lugar, facilitando la toma de decisiones estratégicas basadas en datos.

Está desarrollada con una arquitectura modular, escalable y preparada para crecer.
